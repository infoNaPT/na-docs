# Jitsi
*atualizado a {docsify-updated}*

Este programa está acessível no endereço [Jitsi NA Portugal](https://meet.na-pt.org)

---

Para os participantes de cada reunião online:

- Não é necessário instalar qualquer programa para entrar numa reunião.
- Não é necessária palavra-passe
- apenas usam o endereço da vossa reunião
  - exemplo: https://meet.na-pt.org/qualquercoisa (sendo _qualquercoisa_ o nome da reunião que irão frequentar)

---

Para os moderadores de cada:

- reunião
- comissão
- CSA
- CSRP
- RPHI
- outras

> devem enviar um email para a informática NA Portugal - [clica aqui para enviar](mailto:informatica@na-pt.org)
> para obterem:
 - utilizador
 - palavra-passe

E após instalarem a aplicação adequada para vós [continuar para instalar](#Instalar-apenas-para-moderadores)
